# Sequence
Cross-platform (currently linux and windows) library to identify and handle list of names
having a numerical token as a collection.

# Inspiration
* Clique python module
* rvls utility from TweakSoftware

# Dependencies
* C++11
* cmake-3.5

# Submodules
* fmt-6.0.0
